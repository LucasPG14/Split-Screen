## Controls

 - W, A, S, D to move the first player.
 - UP, DOWN, LEFT and RIGHT arrows to move the second player.
 - I, J, K, L to move the third player.
 - 1, 2, 3, 4 to move the fourth player.
 - Press ENTER to return to the menu.
